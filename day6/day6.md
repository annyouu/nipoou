## Dartの文法勉強(変数宣言)

# 何につまづいたのか・分からなかったこと
1. 修飾子が何か
2. finalとconstとvarの使い分け
3. インスタンスを生成するって具体的に何してんの？

# 理解したことを踏まえて説明する
1. 修飾子は、変数の型定義の前につける接頭語。
    - 修飾子 型 変数名 = 値;とやる。

2. varとconstとfinalの違い
- var: 何度でも代入し上書きする事ができる。
- const: 定数に対する修飾子。コンパイル時(エラーチェック＋値を確定できるものは確定させるフェーズ)で、代入が行われる。値の再代入はできない。1回のみ。
- final: 定数に対する修飾子。varとconstの中間みたいなイメージで、インスタンス生成時に代入がされる。値の再代入はできない。

3. クラスという設計図を元に、メモリ上に実体(データのまとまり)が確保されること

例:
```Dart
void main() {
  final now = DateTime.now();
}
```
DateTime.now()でインスタンス生成されるということは、
メモリにDateTimeオブジェクトが作られるということ。
- final now = ：そのオブジェクトへの参照を一回だけ保持するポインタを得る。
- 変数に入らなければ、ガベージコレクションで消える。

finalは、変数の代入では、一回のみできる。ただ、どのタイミングで代入(参照)が起きるかの違いがこの3つでそれぞれある！！！
インスタンス生成に限った話をすると、ポインタへの参照も一回だけで、それを切り替えることもできないよってことになる。
